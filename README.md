# OpenTelemetry Practice Quiz

A web-based quiz application for the OpenTelemetry Certified Associate (OTCA) exam preparation.

## Usage

**Online:**:  Visit the live quiz at [GitHub Pages](https://hemandee.github.io/open-telemetry-cert-sample-quiz/quiz.html)

## About

This quiz app is based on the official practice questions from the CNCF OpenTelemetry Certified Associate exam [repository](https://gitlab.com/cncf-exams/opentelemetry-certified-associate-otca/-/blob/main/5.%20Practice%20Exam.md)


## Features

- 42 practice questions covering all OTCA exam topics
- One question at a time with immediate feedback
- Progress tracking and scoring
- Detailed explanations for each answer
- Responsive design for desktop and mobile



**Local:**

1. Open `quiz.html` in any web browser
2. Answer questions one by one
3. Get instant feedback with explanations
4. View final score and retake as needed

## Exam Sections Covered

- **Section 1:** Observability Fundamentals (18%)
- **Section 2:** The OpenTelemetry API and SDK (46%)
- **Section 3:** The OpenTelemetry Collector (26%)
- **Section 4:** Maintaining and Debugging Observability Pipelines (10%)

## Source

Questions and answers are sourced from the official CNCF exam materials and are used for educational purposes.
